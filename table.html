Mapping India's Folk Arts

     An idea we have been working on for some time - a start at putting together all of India's rich folk arts in one place in this map. The more time we spent on researching, the more deeply we started thinking how much there is to do, we've only just scratched the surface! This map is work in progress, please write to us if you have suggestions, additions or if we've made any mistakes :). In this map we wanted to focus more on folk arts and not crafts as such (even though for Tripura, Meghalaya, Manipur, Mizoram we could find more crafts and not arts) and the next attempt will also be at mapping crafts.
